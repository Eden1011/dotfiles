# My dotfiles
Place into $HOME/.config/

Important commands:
  - `Mason` for managing LSPs.
  - `<C-e>` to open Neotree.
  - `<C-t>` to open Telescope (look for files).
  - `<C-g>` to open Telescope live_grep (find pattern in all files).
  - `<leader>sb` to find pattern in current file.
  - `<C-l>` to go to right window.
  - `<C-h>` to go to left window.
  - `<leader>be` to open buffer window (on left, switch using `<C-h>`).
  - `<leader>bo` delete all other buffers.
  - `<leader>bd` delete current buffer.
  - `<S-h>` or `[b` to switch to buffer previous buffer.
  - `<S-l>` or `]b` to switch to buffer next buffer.
  - `U` is undo, `u` undoes a `U`.
  - `gd` go to definition.
  - `<C-Down>` to paragraph down.
  - `<C-Up>` to paragraph up.

Additional commands:
  - `<leader>us` toggle spelling check.
  - `<leader>uf` toggle global LSP.
