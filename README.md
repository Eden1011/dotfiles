# My dotfiles
Place into $HOME/.config/

Important commands:
  - `Mason` for managing LSPs.
  - `e 'path'` to open explorer.


### Neovim
Contains LazyNvim preset + transparent theme + Mason and LspConfig.

### Tmux
Changed:
  - Added mouse support.
  - Panes/Windows indexed starting from 1.
  - prefix set to `<C-Space>`.
  - `|` for horizontal split, `-` for vertical.
  - TPM

### Alacritty
Changed font and opacity.
