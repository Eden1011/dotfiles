# DIY Configuration

This directory contains configuration files that need to be manually imported.

## KDE Desktop Shortcuts

**Step 1**: Open KDE System Settings:

```bash
systemsettings # Equivalent of Meta key -> System settings
```

**Step 2**: Navigate to Shortcuts:

- Go to **Workspace** -> **Shortcuts** -> _Look top right corner_.
- Select the file: `diy/kde/kde_desktop_shortcut_scheme.kksrc`.
- Click **Apply**.

## Vimium C Extension

**Step 1**: Install the Vimium C extension:

- Open your browser's extension store.
- Search for "Vimium C" and install it.

**Step 2**: Import keybindings:

-Choose and import file: `diy/vimium/vimium_c-20250827_145959.json`.

Note:

- These configurations must be imported manually as they cannot be automated through dotfiles.
